# 🚀 LaunchPad

**The Ultimate Professional & Academic Ecosystem for Ages 12–18**

LaunchPad is a full-stack social platform built for Gen Z and Gen Alpha — combining mentorship, university planning, micro-internships, freelancing, gamification, and AI-powered admissions counseling into one secure, engaging ecosystem.

## Features

| Module | Description |
|---|---|
| **Dynamic Portfolio** | Multimedia projects, verified skill badges, endorsements, journey timeline |
| **AI Admissions Counselor** | Predictive analytics calculates admission probability + actionable recommendations |
| **Micro-Internships** | 5-hour virtual work simulations from sponsored companies |
| **Freelance Board** | Safe gig marketplace with parent-approved wallet |
| **Mentorship Matrix** | Verified mentors + AI-matched peer cohorts |
| **Gamification** | XP, levels, streaks, daily quests, LaunchPad coins + reward store |
| **Safety Layer** | Age-gated cohorts, AI content moderation, adult DM filter, vetted spaces |
| **Uni Planning** | Scholarship aggregator, campus hubs, application task tracker |

## Tech Stack

- **Frontend:** React 18, Vite, React Router v6
- **Backend:** Node.js, Express, SQLite (better-sqlite3)
- **Mobile:** CapacitorJS (iOS + Android wrappers)
- **Auth:** JWT + bcrypt

## Quick Start

```powershell
# Backend
cd backend
npm install
npm run seed
npm start

# Frontend (separate terminal)
cd frontend
npm install
npm run dev
```

Open **http://localhost:5173**
